# Boucles

## L'écriture d'une boucle avec `while`

	int i = 0;
	
	while(i != 10)
	
	{
	
		...
		
		i++;
	
	}

## L'écriture d'une boucle `do... while`

La boucle consiste à exécuter un code, puis de faire le test. La boucle est par conséquent exécutée une fois.

	int i = 0;
	
	do
	
	{
	
		...
	
	}

	while(i != 10);

## L'écriture d'une boucle `for`

	int i = 0;
	
	for(i = 0 ; i != 10 ; i++)
	
	{
	
		...
	
	}
